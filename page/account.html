API Key Instructions

Create an API key at https://account.arena.net/applications with all permissions checked and paste it in the "API Key" input box above in order to view your GW2 account's data. You may have multiple accounts or API keys, so give them a name to help you remember.

Privacy Statement

Last changed on: 2016-02-01

Knowing an API key allows anyone to view your account's ingame information, such as: characters and guilds, bank and inventory contents, gold and assets, achievements and unlocks, Trading Post and PvP match history, and playtime and activity. You may limit which of these type of information is exposed when creating an API key, and disallow further access to these information by deleting the API key at the ArenaNet website.

GW2Timer.com does not view, store, collect, monitor, or datamine API keys or your account information. When you enter the API key into the website, you will be downloading data from ArenaNet servers and performing calculations using this web application locally on your computer, and the keys are stored in your web browser, not a server.

GW2Timer.com uses Google Analytics and is able to view and store text that is shown on your address bar when you are on this website. If you do not wish to expose your API key, do not put your API key in the browser address bar. To share your key with other people, private message them the key and direct them to gw2timer.com/account

Console

          Search by Keyword

          The search bar allows you to search for items using space-separated keywords. It will look through every item's tooltip for any match of the words you type. For example, typing: "exo arm" will filter in any exotic armor pieces you have. If you want to look for an item that has toughness stats, then just type "toug" for partial matching of items with that keyword. You can search by item type by matching the words at the bottom of items' tooltips, such as "gizmo" (toys), "consumable" (foods), or "container" (bags).

          Search by Price

          Filter in items that are tradeable with a price by first typing either "<" or ">" then a period-separated coin string. For example, to search for items that can be sold for more than 1 gold on the Trading Post, type in the search ">1..". If you type ">1" then items valued above 1 silver will be shown. Here is the format for coin strings:

          0.1 = 10
          .01 = 1
          1 = 1
          1.55 = 1 55
          2.4.7 = 2 4 7
          2..6 = 2 0 6
          230.6 = 2 30 60
          920.. = 920

          Notes

          Coin price displayed over the tabs are sum of that tab's tradeable items' value (after 5% listing fee and 10% tax) if they had been listed or sold instantly on the Trading Post. Price above the bank is the sum of all tradeable items' prices. Double click an item to open its wiki page.

          Wardrobe Usage

          For fast page load, item icons and data are not downloaded and calculated until you have opened a tab to view that category of skins. Each icon shown is actually the item that unlocks that skin. Multiple items can have the same skin, so representative items were chosen based on these criteria: the item is most abundant and relatively cheap on the Trading Post, it should not be bindable so you can resell it after unlocking it, and for bound items it should be the most basic tier as to be cheaper from karma or cultural vendors.

          Click on an item icon to use the context menu for ways to obtain the item. The coin price shown is the current Trading Post sell listing, which may be cheaper if you use buy orders instead.
